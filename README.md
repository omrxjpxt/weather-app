# 🌦 Weather App

A modern and responsive weather application built using React that fetches real-time weather data using the Open-Meteo API.

## 🚀 Features

* 🔍 Search weather by city, district, or local area
* 🌡 Real-time temperature, humidity, and wind speed
* 📅 3-day weather forecast
* 🎨 Clean and responsive UI
* ⚡ Fast API-based data fetching

## 🛠 Tech Stack

* React (Vite)
* Open-Meteo API
* JavaScript, CSS
